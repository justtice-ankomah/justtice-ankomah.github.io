    ON THE SERVER

socket.emit("message", "something...")  => send message to only single person who connect to the server. eg. "John welcome to the chat room"

socket.broadcast.emit("message", "something..."); => send message to all those who are connected to the server except the one who is connecting. eg "members new user john has connected to the chat room"

io.emit("message", "something...") => send message to all the client who has connected to the server. eg. "good monning all members"
